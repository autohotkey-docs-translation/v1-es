Empezando

Antes de que AutoHotkey pueda hacer algo, debe escribir un script que le diga qué hacer.
Para saber cómo, lea el tutorial o empiece directamente.

Básicamente: crear un archivo de secuencia de comandos → agregar algunas teclas de acceso rápido → ejecutar el script.

También puede obtener ayuda o encontrar muchas secuencias de comandos útiles en nuestro amistoso foro de discusión.

¿Por qué estoy viendo esto?

Ejecutó AutoHotkey sin diciéndolo decir que archivo de script ejecutar, y no pudo encontrar un script en ninguno de los sitios habituales.
